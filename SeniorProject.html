Senior Project

This project was for CST-451 and CST-452, part of the Bachelor of Science in Software Development at Grand Canyon University. The purpose of this project was to demonstrate planning and design capabilities.


Project Overview

Problem and Project Background

The purpose of this project is to develop an online game application that allows users to play a puzzle game where internet access is available. The goal is to provide entertainment for users by developing an application that is easy to access and use.

Application of Christian Worldview

Project Objectives

The objective of this project is to build a web application that can be accessed by users. The users will be able to log in to play the game and can save the games they are playing to their logged-in account.

Project Challenges and Measures of Success

The project will meet six deadlines during its development. The project\’s success is measured by what percentage of goals had been met before each deadline.

Benefits and Opportunities

This project will allow all participating developers to gain experience in the web application development field and demonstrate knowledge of common development practices.

Project Summary

This project is a website where users can register and login to play a game based on Lights Out. The user can choose to start a new game, save a game in progress, and resume a saved game at any time. Both user login information and saved game data are stored securely in a relational database.

This diagram is of the application’s sitemap, showing what parts of the application can be accessed through other parts of the application. The game can only be accessed after creating a user and logging in, otherwise, the user will be redirected to the registration page.


This image details the program flow for the application. When the user first accesses the website, they are allowed to either login or register as a new user account. If the user attempts to access the game without either logging in first or creating an account, they will be redirected to the User Registration page. After registering or logging in, the user will be allowed to access and play the game. While they are playing the game, the user has three options. They can choose to save the game which saves the game to a MySQL database, they can choose to load one of their saved games from a list of saved games retrieved from the MySQL database, or they can choose to restart the game with a new puzzle to solve.
