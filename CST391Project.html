Milestone Project for Grand Canyon University CST-391

Project Summary

This project was for CST-391, part of the Bachelors of Science in Software Development at Grand Canyon University. The purpose of this project was to demonstrate planning and design capabilities as well as showcase experience in React, Angular, and Express.

This product is a set of websites that simulate an inventory database for a fictional company. There are two front end applications for the database, one written using React, the other written using Angular. The back end application stores all of its information in a MySQL database and communicates using a REST API.

Each version of the front end application can add new entries to the database, edit previous entries in the database, or remove entries from the database. When creating or editing entries in the database, the user must complete a form with all of the required information.
